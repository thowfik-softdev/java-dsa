# 04-weekly/ — 🗓️ Weekly reviews

**Why this folder exists:** once a week (no new topics) we consolidate. Mixed, **interleaved** practice across the week's patterns — jumbled like a real interview — which exposes the gaps that day-to-day "blocked" practice hides.

- One file per week: `YYYY-Www.md` from [06-templates/weekly.md](../06-templates/weekly.md).
- Updates mastery in [../05-progress/PROGRESS.md](../05-progress/PROGRESS.md) and re-plans next week in [../ROADMAP.md](../ROADMAP.md).
- Flags weak topics to reinforce *before* the roadmap advances.
