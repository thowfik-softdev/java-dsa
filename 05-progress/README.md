# 05-progress/ — 📊 The dashboard

**Why this folder exists:** one file, [PROGRESS.md](PROGRESS.md), holds everything about *how you're doing* — streak, stats, mastery levels, weak topics, and the spaced-repetition due-list. Deliberately ONE file (not the 5-file sprawl of bigger systems) so it never falls out of sync.

The Mentor updates it every session. You read it when you want the honest picture.
