# 01-concepts/ — 📘 Your permanent notebook

**Why this folder exists:** this is your forever DSA textbook. One deep note per topic, written *by you + the Mentor* as you learn it — so months later you re-read *your own* explanation, not a stranger's.

- One concept per file, `kebab-case.md` (e.g. `two-pointers.md`).
- Created on demand: when you say **"generate notes"** at the end of a session.
- Structure is fixed (see [06-templates/concept.md](../06-templates/concept.md)): intuition → why it exists → how it works → Java → **JS comparison** → brute force → optimal → complexity → mistakes → interview tips → related → practice problems → next problem.
- Linked with `[[wikilinks]]` so Obsidian builds a graph of your knowledge.
