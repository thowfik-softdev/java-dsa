# 02-problems/ — 🧩 Solved problems log

**Why this folder exists:** every problem you solve becomes a permanent entry — the problem, *your* approach, the solution, what you learned, and a pointer to the next problem to try. This is what you revise from and what proves your volume for FAANG (~300–500 over the journey).

- One file per problem: `lc-<num>-<slug>.md` or `nc-<num>-<slug>.md`.
- Mirrors runnable code in [../src/](../src/).
- Structure: see [06-templates/problem.md](../06-templates/problem.md).
- Each entry ends with a **Next problem** pointer so you always know what to attempt next.
