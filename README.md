# Java DSA Mastery OS

A personal system to take me from **complete beginner → FAANG-ready Java DSA**, sustainably, at 2 hours/day.

One **Mentor** (no separate agents) that teaches, coaches, tests, and documents — guided by two files and a notebook that I keep forever.

## Start here
1. **[CONTEXT.md](CONTEXT.md)** — the master memory. *If a chat dies, a new chat reads this and resumes.*
2. **[RULES.md](RULES.md)** — how the Mentor teaches (the daily flow, hint ladder, note structure).
3. **[ROADMAP.md](ROADMAP.md)** — the full zero→FAANG curriculum.

## Each session (2 hrs)
**Recall → Learn → Practice → Solve → "generate notes" → Log.** (Recall-first = retention.)

To start a session in a fresh chat: paste [RULES.md](RULES.md). The Mentor boots by reading [CONTEXT.md](CONTEXT.md) and the latest [03-daily/](03-daily/) log, then begins.

## Map
```
CONTEXT.md     master memory (resume point)
```

Every folder has a README explaining why it exists.

## Built for
Obsidian (wikilinks, graph) + Git (history, streak). Learner knows JavaScript/Node — everything is taught by comparison to JS.
