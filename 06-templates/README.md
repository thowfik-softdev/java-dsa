# 06-templates/ — 📄 Note shapes

**Why this folder exists:** the fixed structures the Mentor fills in, so every note looks the same and is skimmable/searchable forever.

- [concept.md](concept.md) — the notebook entry (`01-concepts/`)
- [problem.md](problem.md) — a solved problem (`02-problems/`)
- [daily.md](daily.md) — a day log (`03-daily/`)
- [weekly.md](weekly.md) — a weekly review (`04-weekly/`)

You never edit these directly during a session — the Mentor copies them.
