# 03-daily/ — 📅 Daily logs

**Why this folder exists:** one log per study day. It records what we did so that (a) tomorrow's **recall** has a source, and (b) any fresh chat can read the latest log and know exactly what just happened.

- One file per day: `YYYY-MM-DD.md` from [06-templates/daily.md](../06-templates/daily.md).
- The Mentor fills it during the LOG step; you add a one-line reflection.
- The most recent file here is the second thing a new chat reads (after [CONTEXT.md](../CONTEXT.md)).
